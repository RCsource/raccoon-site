Raccoon

A programming language designed for simplicity.

What is Raccoon?

Raccoon is a simple programming language designed to be easy to learn and use. It is perfect for beginners who want to start learning programming.

Why choose Raccoon?

Raccoon is a programming language designed for simplicity and ease of use. It aims to provide a clean and intuitive syntax that is easy to read and write, even for beginners. Features:

  • Simple and easy to learn syntax
  • Functions and procedures
  • Conditionals and loops
  • Arrays and dictionaries
  • I/O operations
  • Own packages manager
